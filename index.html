Bypass The Filter

Introduction

bypassthefilter.org is an community project to list what filters may be used in your country of residence (or as a visiting tourist), and how one might bypass the filter.

Types of filters

There are a number of different types of internet filters, such as:

  • Desktop installed software, such as adult content filters, anti-virus or China's Green Dam.
  • Desktop or local area network DNS server which will not return the correct IP address for content which is restricted.
  • Web browser configured (or WPAD) HTTP proxy server such as squidGuard to prevent delivery of content to the user.
  • In-line upstream content filtering appliance which blocks TCP/IP packets which contain filtered content.
  • Internet Service Provider based content filtering
  • Government / State based filtering on a national or international level
